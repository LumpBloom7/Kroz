Kroz

A text-based adventure game written in C++.

View on GitHub Download .zip Download .tar.gz

What is this

This is my attempt to create a text-based hacker simulator kind of game. It's not that good, but I'm learning as I'm creating this. This is a console project created using C++. If it becomes good enough I'll maybe open-source it.

Features

Currently implemented:

  • Main Menu (Just for show, doesn't do anything)
  • Clear console function
  • People factory
  • Cross-platform compatibility(Confirmed on Linux & Windows)
  • Partial savegame system for NPC's
  • Somewhat working console functionality(to control game variables)
  • Game cinematics

Next up:

  • Revamped relationships mechanic
  • Cross-platform make directory function(for savegames)
  • Working vocabulary system
